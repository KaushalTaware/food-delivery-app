# E-Commerce Frontend (React)

This is the frontend part of an e-commerce web app built using React.  
Users can browse products, view details, and manage their shopping cart.  
Backend and payment integration will be added later.

---

## 🚀 Features

- Responsive product listing UI
- Product detail page
- Add to cart functionality
- React routing
- Clean component-based structure

---

## 🛠️ Tech Stack

- React
- Tailwind CSS (Custom As per Need)
- React Router
